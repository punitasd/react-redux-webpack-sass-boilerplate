# React+Redux+Webpack+Sass boilerplate

A ready to use boilerplate for React Apps.

## Run in dev

* Clone the repo
* Checkout `git checkout develop`
* Change directory `cd src`
* Run `npm install`
* Run `npm run start-dev`

## Run in prod

* Clone the repo
* Checkout `git checkout develop`
* Change directory `cd src`
* Run `npm install`
* Run `npm run build`
* Run `npm start`
